## Ejecutar Localmente

Clonar el proyecto

```bash
  git clone Github
```

Ir a la carpeta clonada

```bash
  cd my-project
```

Instalar las dependencias

```bash
  pnpm install
```

Ejecutar el servidor

```bash
  pnpm run dev
```
