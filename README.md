## Ejecutar Localmente

Clonar el proyecto

```bash
  git clone Github
```

Ir a la carpeta clonada

```bash
  cd my-project
```

Instalar las dependencias

```bash
  pnpm install
```

Ejecutar el servidor

```bash
  pnpm run dev
```


## Stack

**Frontend:** Nextjs, TypeScript, TailwindCSS


## Produccion

Para implementar este proyecto en produccion, ejecute

```bash
  pnpm run build
```

```bash
  pnpm run start
```
